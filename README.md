# Smart-Irrigation-System
Smart Irrigation System that automatically monitors soil moisture and controls water supply to plants using sensors and a microcontroller. The system optimizes water usage, reduces manual effort, and helps maintain healthy plant growth.
The Smart Irrigation System is an automated solution designed to efficiently manage watering of plants using real-time soil moisture data. The system uses sensors to detect the moisture level in the soil and automatically activates or deactivates the irrigation pump when needed.
This helps conserve water, reduce human intervention, and ensure plants receive the optimal amount of water.
## Features
- Automatic watering based on real-time soil moisture.
- Saves water by preventing over-irrigation.
- Low-cost and easy-to-build solution.
- Real-time monitoring of soil conditions.
- Can be enhanced with IoT and mobile notifications.
## Components Used
- Arduino / ESP32 / ESP8266 microcontroller.
- Soil moisture sensor.
- Relay module.
- Water pump.
- Power supply.
- Connecting wires.
## Applications
- Home gardening.
- Smart agriculture.
- Greenhouses.
- Water conservation systems.
